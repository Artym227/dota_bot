# dota_bot

Это приватный репозиторий для разработки и автоматического развертывания личного Telegram-бота.

## 🚀 Основные функции
* **Функция 1**: Краткое описание (например, уведомления о погоде).
* **Функция 2**: Описание (например, парсинг данных с сайта X).
* **CI/CD**: Автоматический деплой через GitHub Actions.

## 🛠 Технологический стек
* **Language:** Python 3
* **Library:** `aiogram`
* **Database:** PostgreSQL (если есть)
* **Infrastructure:** GitHub Actions, Docker
